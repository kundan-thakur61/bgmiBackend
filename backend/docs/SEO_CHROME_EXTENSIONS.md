# Chrome Extensions for SEO Productivity

## Essential SEO Extensions for BattleZone Development

### 1. **Lighthouse**
- **Purpose**: Audit performance, accessibility, SEO, and best practices
- **Usage**: Right-click → Inspect → Lighthouse tab
- **Key Metrics**: 
  - Performance Score
  - SEO Score
  - Core Web Vitals (LCP, FID, CLS)

### 2. **Detailed SEO Extension**
- **Purpose**: On-page SEO analysis
- **Features**:
  - Meta tags validation
  - Heading structure
  - Image alt text check
  - Schema markup detection

### 3. **SEOquake**
- **Purpose**: Real-time SEO metrics
- **Features**:
  - Page analysis
  - Internal/external links
  - Keyword density
  - Social metrics

### 4. **Schema Validator**
- **Purpose**: Validate structured data
- **Usage**: Check schema.org markup on any page
- **URL**: https://validator.schema.org/

### 5. **Web Developer**
- **Purpose**: Disable CSS, images, JavaScript for testing
- **Features**:
  - View responsive design
  - Validate HTML/CSS
  - Check page information

### 6. **META SEO Inspector**
- **Purpose**: Inspect meta tags and structured data
- **Features**:
  - View all meta tags
  - Check Open Graph tags
  - Twitter Card validation

### 7. **Redirect Path**
- **Purpose**: Check HTTP redirects
- **Features**:
  - 301/302 redirect detection
  - Server header information

### 8. **PageSpeed Insights**
- **Purpose**: Google's official speed testing tool
- **URL**: https://pagespeed.web.dev/

## How to Use for BattleZone

1. **Run Lighthouse on key pages**:
   - Homepage
   - Tournament listing
   - Match details
   - User profile

2. **Check Schema with Detailed SEO**:
   - Verify tournament schema
   - Validate breadcrumb markup
   - Check FAQ schema

3. **Monitor with SEOquake**:
   - Track keyword optimization
   - Analyze competitor pages
   - Check mobile-friendliness

4. **Validate Schema**:
   - Test all structured data
   - Use Google Rich Results Test
   - Fix validation errors

## Performance Targets

- **LCP (Largest Contentful Paint)**: < 2.5s
- **FID (First Input Delay)**: < 100ms
- **CLS (Cumulative Layout Shift)**: < 0.1
- **Performance Score**: > 90
- **SEO Score**: 100
