# Backlink Outreach Tracker

## Template for Tracking Outreach Campaigns

### Columns to Track:
1. Target Site
2. Domain Authority
3. Contact Email
4. Outreach Date
5. Template Used
6. Response Date
7. Status (Pending/Accepted/Rejected)
8. Link Acquired (Yes/No)
9. Link URL
10. Anchor Text
11. Follow-up Date
12. Notes
